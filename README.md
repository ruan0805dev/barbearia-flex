## Descrição do caso escolhido
O projeto consiste em um site institucional para uma barbearia fictícia chamada **Barbearia FLEX (Barbershop FLEX)**. A ideia é representar um pequeno negócio do ramo de estética masculina, oferecendo informações sobre serviços como cortes de cabelo, barba e combos especiais, além de facilitar o contato com clientes.

---

## Necessidade identificada
Foi identificado que muitos pequenos negócios, como barbearias locais, não possuem presença digital organizada ou sites modernos para divulgação. Assim, o projeto busca resolver essa falta de visibilidade online, facilitando o acesso às informações e promovendo os serviços de forma mais profissional.

---

## Público-alvo
O público-alvo do site são homens jovens e adultos que procuram serviços de barbearia, como cortes modernos, cuidados com barba e atendimento personalizado. O site também atende clientes que desejam agendar horários de forma rápida e prática.

---

## Objetivo do site
O principal objetivo do site é apresentar os serviços da barbearia, transmitir profissionalismo e facilitar o contato com os clientes por meio de agendamento via WhatsApp.

---

## Processo de desenvolvimento
O desenvolvimento foi realizado em etapas: primeiro foi criada a estrutura HTML do site, depois aplicada a estilização com CSS e Bootstrap para melhorar o design e a responsividade. Em seguida, foram adicionadas funcionalidades em JavaScript para interação com o usuário, como o botão de agendamento e a exibição do horário de funcionamento. Durante o processo, ajustes foram feitos para melhorar o layout do banner, organização das imagens e experiência do usuário.

---

## Principais decisões do projeto
- Uso de estrutura semântica no HTML (header, section, footer)
- Utilização do Bootstrap para responsividade e componentes prontos
- Criação de um banner (hero section) com imagem de fundo e destaque visual
- Aplicação de cores escuras com verde como destaque para identidade visual moderna
- Organização do conteúdo em seções claras (início, sobre, serviços e contato)
- Uso de animações simples para melhorar a experiência do usuário
- Implementação de layout responsivo para dispositivos móveis

---

## Funcionalidade em JavaScript
Foi implementada uma funcionalidade de interação com o usuário através do JavaScript:

- Botão “Agendar horário” que abre diretamente o WhatsApp da barbearia
- Botão “Mostrar horário” que exibe ou oculta os horários de funcionamento
- Animação de elementos ao rolar a página, melhorando a experiência visual

Essas funcionalidades foram escolhidas para tornar o site mais dinâmico e facilitar o contato entre cliente e barbearia.

---

## Uso de Bootstrap
O Bootstrap foi utilizado para agilizar o desenvolvimento e garantir responsividade. Foram usados componentes como:
- Navbar responsiva
- Grid system (colunas para serviços e sobre)
- Botões estilizados
- Cards para exibição dos serviços

O uso do Bootstrap permitiu um layout moderno e adaptável a diferentes tamanhos de tela.
